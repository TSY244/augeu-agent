## 加载模式

加载模式和从哪里加载信息相关

如果使用本地模式的话，将会尝试从本地读取rule.json 文件（可以指定）

如果是远程模式的话，需要给一个远程的url 指向远程rule.json

如果是basic 加载方式，那就会使用本程序默认的加载方式
